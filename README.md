# MeghDoot – Smart Weather Forecast Platform 🌦️

MeghDoot is a modern weather forecasting web application that provides real-time weather data, forecasts, and interactive weather insights for cities across the world.

The application uses the OpenWeather API to fetch live weather data and presents it with a beautiful animated UI inspired by modern weather dashboards.

---

## 🌍 Live Demo

https://meghdoot-weather-app.vercel.app

---

## ✨ Features

• Real-time weather data for any city worldwide
• 5-day weather forecast
• Hourly temperature preview
• Animated weather icons and effects
• Sunrise and sunset visualization
• Temperature trend graph
• Dynamic UI based on time of day (dawn / day / dusk / night)
• Rain animation during rainy conditions
• Temperature unit toggle (°C / °F)
• Beautiful glassmorphism UI design
• Responsive design for desktop and mobile
